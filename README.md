# Meu Kanban Next.js

Meu primeiro projeto com GitHub, Codex e Vercel.

Aplicativo Kanban em Next.js para gestao de tarefas, projetos e melhoria continua.

## Funcionalidades

- Criar colunas
- Editar colunas
- Excluir colunas
- Criar tarefas
- Editar tarefas
- Excluir tarefas
- Arrastar tarefas entre colunas
- Salvar dados no navegador

## Tecnologias

- Next.js
- React
- TypeScript
- Tailwind CSS
- LocalStorage

## Como rodar

npm install
npm run dev

Abra no navegador:
http://localhost:3000

## Deploy

Projeto preparado para publicar na Vercel.
